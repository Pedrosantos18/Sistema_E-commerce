# E-commerce em Java (Console)

# Descrição
Este projeto é um sistema de e-commerce desenvolvido em Java, totalmente orientado a objetos. Ele simula a experiência de compra em um comércio online.

# O sistema permite:

- Criar e gerenciar um carrinho de compras;
- Adicionar produtos físicos e digitais com cálculo automático de preço final (peso, taxas, etc.);
- Remover produtos do carrinho;
- Visualizar o catálogo completo de produtos e os itens do carrinho;
- Finalizar compras utilizando diferentes **formas de pagamento** (Pix, Boleto e Cartão de Crédito com parcelamento);
- Aplicar descontos e taxas automaticamente nos pagamentos;
- Interface simples de **menu interativo em console**.

# Tecnologias e conceitos usados

- Java
- Programação Orientada a Objetos (POO).
- Herança e Polimorfismo.
- Interfaces.
- Listas (Arrayslist) e loops.
- Formatação de saída com printf.
